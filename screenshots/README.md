# Screenshots Directory

## How to Add Screenshots

1. **Take Screenshots**:
   - **Streamlit UI**: Open your app at http://localhost:8501 and take a screenshot
   - **Code Structure**: Take a screenshot of your project in VS Code

2. **Save Screenshots**:
   - Save UI screenshot as: `streamlit_ui.png`
   - Save code screenshot as: `code_structure.png`
   - Place both files in this `screenshots/` directory

3. **Recommended Screenshot Content**:

   ### streamlit_ui.png should show:
   - Main interface with symptom input box
   - Patient profile selector (Adult/Infant)
   - Analysis results with risk assessment
   - Medical guidance response
   - Doctor report tab

   ### code_structure.png should show:
   - File explorer showing project structure
   - One or two key code files open (e.g., main_agent.py, app.py)
   - Clear view of the modular architecture

4. **Screenshot Tips**:
   - Use high resolution (at least 1920x1080)
   - Ensure good contrast and readability
   - Crop out unnecessary borders
   - Consider using tools like:
     - Windows: Snipping Tool or Win+Shift+S
     - Mac: Cmd+Shift+4
     - Browser extensions: Awesome Screenshot, Nimbus

5. **Once added, the README will automatically display them**
